## 📌 프로젝트 설계
### 비즈니스 요구 사항
1. 회원

- 회원 가입, 조회
- 회원은 일반과 VIP
- 자체 DB구축할 수도 있고, 외부 시스템과 연동할 수도 있음

2. 주문과 할인 정책

- 회원은 상품 주문 가능
- 회원 등급에 따라 할인 정책 적용 가능
- 할인 정책은 변경 가능성이 높음

> 🐺 추후 변경될 사항들이 매우 많음, 인터페이스를 만들고 구현체를 갈아끼우는 방식으로 설계해야 함
